  Cafe POS Print Bridge
  เชื่อมเครื่องปริ้นใบเสร็จ EPSON TM-T82X เข้ากับ
  เว็บ cafe-pos-sable.vercel.app
══════════════════════════════════════════════════

ก่อนติดตั้ง
───────────────────────────────────────────────────
1. ต่อเครื่องปริ้น EPSON TM-T82X เข้ากับ PC ผ่านสาย LAN
   (จะต่อตรงเข้า PC หรือผ่าน router ก็ได้)

2. เปิดเครื่องปริ้น รอจนไฟติดนิ่ง


วิธีติดตั้ง (ทำครั้งเดียว)
───────────────────────────────────────────────────
1. คลิกขวาที่ไฟล์ install.bat
2. เลือก "Run as administrator" → Yes
3. รอประมาณ 10 วินาที
4. หน้าจอจะขึ้นว่า "ติดตั้งสำเร็จ!"
5. เปิด browser ไปที่ https://cafe-pos-sable.vercel.app


หลังติดตั้ง
───────────────────────────────────────────────────
- Bridge ทำงานเป็น Windows Service ชื่อ "CafePosBridge"
- เปิดอัตโนมัติทุกครั้งที่บูตเครื่อง
- ไม่ต้องเปิดหน้าต่าง command line ไว้
- ถ้า crash จะ restart ตัวเองภายใน 3 วินาที


หาเครื่องปริ้นอัตโนมัติ
───────────────────────────────────────────────────
ครั้งแรกที่เปิด bridge จะสแกนหาเครื่องปริ้น EPSON
ในเครือข่ายเอง (port 9100) — ใช้เวลา 10-30 วินาที

ถ้าหาไม่เจอ:
- ตรวจสอบว่าเปิดเครื่องปริ้นแล้ว
- ตรวจสอบสาย LAN ต่อแน่นทั้งสองด้าน
- ลอง restart service ที่ services.msc


ปัญหาที่พบบ่อย
───────────────────────────────────────────────────

▸ เว็บขึ้น "ออฟไลน์" / "bridge ไม่ตอบ"
  → เปิด services.msc หา "CafePosBridge" → Right-click → Start
  → หรือ restart PC

▸ เว็บเชื่อมไม่ได้แต่ service Run อยู่
  → ดู log: C:\ProgramData\cafe-pos-bridge\bridge.log
  → ตรวจสอบ firewall ว่าไม่ blockport 8080

▸ พิมพ์ภาษาไทยเพี้ยน
  → เครื่องปริ้นต้องรองรับ TIS-620 (รุ่น TM-T82X ของไทยรองรับมาตรฐาน)

▸ ปริ้นเตอร์รุ่นอื่น
  → รองรับเฉพาะ EPSON TM ที่ใช้ ESC/POS + TIS-620 ผ่าน port 9100


การจัดการ service
───────────────────────────────────────────────────
- หยุดชั่วคราว:  ใน services.msc → Right-click "CafePosBridge" → Stop
- เปิดอีกครั้ง:    ใน services.msc → Right-click → Start
- ลบทิ้ง:        คลิกขวา uninstall.bat → Run as administrator


การ update เป็นเวอร์ชันใหม่
───────────────────────────────────────────────────
1. รัน uninstall.bat (Run as administrator)
2. ลบโฟลเดอร์เก่าทิ้ง (ถ้าเหลือ)
3. แตก zip เวอร์ชันใหม่
4. รัน install.bat (Run as administrator) อีกครั้ง
